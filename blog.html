Starting My SEO Practice: What I'm Learning First

I decided to learn SEO before moving into social media marketing, because I wanted one skill built completely before starting the next. Instead of only reading about SEO, I built this website as a live practice project — every change here is a real experiment.

Why SEO first

SEO forces you to think about structure: how a page is organized, what a heading actually does for both readers and search engines, and why a meta description matters even though most visitors never see the raw HTML behind it.

What I'm focusing on right now

  • Writing clear, keyword-relevant headings (H1, H2, H3) instead of vague ones.
  • Keeping meta titles and descriptions accurate and specific to each page.
  • Structuring internal links so pages connect logically (Home → Blog → Projects → Contact).
  • Preparing to connect Google Search Console and Analytics, so I can see real search performance instead of guessing.

What's next

Once this site is fully optimized and connected to real analytics data, I'll move on to keyword research using Google Keyword Planner, followed by a technical SEO pass — checking page speed, mobile usability, and crawlability.
